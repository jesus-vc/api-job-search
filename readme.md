# Jobly Application

## Description

- This is a pure API app built using Node, Express, PostgresSQL.
- The app itself is a job-searching API consisting of 3 routes: /users, /jobs, /companies.
- A 4th route (/auth) exists to handle authentication.

## Getting Started

- Run application:
  node server.js

## Testing

- Run all tests:
  jest -i
